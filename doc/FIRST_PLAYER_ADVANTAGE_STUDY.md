# Bao la Kiswahili 先攻・後攻差の調査

Version: 0.2.1
更新日: 2026-07-15

位置づけ: 標準初期局面の1,000局調査を記録した履歴文書。後続調査と現在の計画は`FIRST_PLAYER_ADVANTAGE_RESEARCH.md`を参照する。

## 1. 目的

この文書は、このリポジトリのコンピュータ対戦結果を用いて、Bao la Kiswahiliに先攻・後攻の有利不利が観測されるかを調べた記録である。

ここで扱うのは、Bao一般について先手必勝・後手必勝を証明する理論研究ではない。このルールエンジン、AI、探索条件で実施したAI対AI対局から、席順による偏りと実装上の再現性を調べる実験研究である。

## 2. 用語

この実装では盤面側を`South`と`North`で表す。初期状態では`state.player`が指す側から対局が始まるため、本調査では初手を指す側を先攻として扱う。

ただし、保存済みベンチマークとの対応を明確にするため、集計値は`South勝利`、`North勝利`と表記する。

## 3. 予備調査

### 3.1 条件

保存済みのPhase 0基準ベンチマーク100局を、先後差の観点から再解析した。

```sh
node tools/benchmark.js --games 100 --seed 20260706 \
  --first hard --second normal \
  --first-profile legacy --second-profile legacy \
  --time-limit 0 --max-depth 2
```

| 条件 | 値 |
| --- | --- |
| 対局数 | 100局 |
| first / second | hard / normal |
| 評価profile | legacy |
| 最大深度 | 2 |
| 時間制限 | なし |
| 開局手数 | 0 |

### 3.2 結果

| 指標 | 結果 |
| --- | ---: |
| South勝利 | 48局 |
| North勝利 | 52局 |
| 引き分け | 0局 |
| South勝率 | 48.0% |
| South勝率の95% Wilson信頼区間 | 38.5%–57.7% |

100局ではNorthが52対48で4局多く勝ったが、信頼区間に50%が含まれるため、明確な先後差は検出されなかった。

> 予備調査では、先攻または後攻の統計的優位を確認できなかった。

## 4. 同一AIによる1,000局調査

### 4.1 実行方法

GitHub Actionsの`First-player advantage study`を使用した。最初の構成では1ジョブ内で1,000局を順次実行したため30分で時間切れとなった。その後、50局ずつ20バッチへ分割し、最大5バッチを並列実行する構成へ変更した。

各バッチは25個のペア対局を含む。`tools/benchmark.js`は同じ開始局面を2局で共有し、`first`と`second`をSouth/Northへ交互に配置する。

```text
第1局: first = North、second = South
第2局: first = South、second = North
```

### 4.2 条件

| 総対局数 | 1,000局 |
| バッチ数 | 20 |
| 1バッチ | 50局 |

| 探索方式 | phase2 対 phase2 |

| 時間制限 | なし |
| 最大手数 | 300 |
| 開局手数 | 0 |
| 開始局面 | 標準初期局面 |

## 5. 結果の解釈

### 5.1 強い再現性は確認された

異なる20個のseedと20個の独立したActionsジョブでも、勝敗と平均手数が完全に一致した。少なくとも、この実装と探索条件においてNorth勝利は偶発的な1局ではなく、非常に再現性の高い結果である。

この結果は、次のいずれか、または複数が存在する可能性を示す。

- 標準初期局面に強いNorth側の優位がある。
- 深度2の同一AI同士では、決定的な同一手順へ収束する。
- 評価関数や合法手の列挙順が、特定の座席に有利な着手系列を生む。
- 乱数seedが実際の着手選択へ影響しない条件になっている。
- ルール実装または初期手番の扱いに、意図しない非対称性がある。

### 5.2 1,000局を1,000個の独立標本とは扱えない

今回のすべての対局は`opening-plies 0`であり、同じ標準初期局面から始まった。同一AI・固定深度・時間制限なしという条件では、着手選択がほぼ決定的になる。

全20バッチで勝敗だけでなく平均手数まで48.0手で一致したことから、実質的には同一または極めて近い対局系列が繰り返された可能性が高い。

したがって、Wilson信頼区間は実行結果の二項集計としては計算できるものの、局面多様性を持つ1,000個の独立標本に対する不確実性として解釈してはならない。

### 5.3 「Baoは後攻必勝」とは結論できない

今回の結果から直接いえるのは、限定された実験条件におけるNorth側の完全勝利である。

次の主張はまだできない。

- Bao la Kiswahiliは理論的に後攻必勝である。
- 熟練者同士でもNorthが必ず勝つ。
- より深い探索でも同じ結果になる。
- ランダムなnamuaまたはmtaji局面でもNorthが有利である。
- 一般的なBaoルールでも同じ結果になる。

## 6. 予備調査との違い

予備調査ではhard対normalを交互に座席へ割り当て、South 48勝・North 52勝だった。一方、本調査では同一hard AI同士が標準初期局面から対局し、Northが1,000局すべてに勝った。

この差は、単純に「対局数を増やしたため」だけでは説明できない。主な相違点は次のとおりである。

- AIの組み合わせがhard対normalからhard対hardへ変わった。
- 評価profileがlegacyからbaoへ変わった。
- 同一AI同士では対局手順が決定的に収束しやすい。

したがって、2つの結果は相互に矛盾するというより、AI構成と評価profileに強く依存することを示している。

## 7. 現時点の結論

現時点で最も正確な表現は次のとおりである。

> このリポジトリの標準初期局面において、同一hard AIを固定深度2で自己対局させると、Northが48手で勝つ決定的な対局系列が再現された。1,000局すべてがNorth勝利だったが、同一局面・同一系列の反復である可能性が高いため、Bao一般の後攻優位や後攻必勝を証明する結果ではない。

この結果は「先後差がない」という予備的な見方を修正する重要な観測である。同時に、ゲーム固有の先後差と実装・AI固有の偏りを分離する追加調査が必要である。

## 8. 次の調査

この節は本調査を記録した時点の課題一覧である。8.1と8.3の診断結果は`artifacts/first-player-study/diagnostics.json`、8.2の後続結果は`FIRST_PLAYER_RANDOM_OPENING_STUDY.md`、現在の未完了実験は`FIRST_PLAYER_ADVANTAGE_RESEARCH.md`に記録している。

### 8.1 対局系列の同一性確認

各局について着手列または局面ハッシュを保存し、20バッチの対局が同一系列かを確認する。

最低限、次を保存する。

- 各手のプレイヤー
- 選択した穴と方向
- phase
- 盤面ハッシュ
- 勝者と終了理由
- 総手数

### 8.2 ランダム開局による局面多様化

標準初期局面だけでなく、同じランダム開局を共有するペア対局を実施する。

```sh

  --first hard --second hard \

  --opening-plies 8 --opening-phase namua \
```

### 8.3 探索深度の比較

深度1、2、3、4で勝者と着手系列を比較する。深度によって結果が反転する場合、観測された偏りは浅い探索に固有である可能性が高い。

### 8.4 実装上の対称性監査

次をコードとテストで確認する。

- 初期手番が意図した側になっているか。
- South/Northを反転した鏡像局面で合法手が対応するか。
- 評価関数が座席反転に対して符号対称か。
- 合法手列挙順が一方の座席だけを優先していないか。
- 捕獲入口、方向、nyumba位置の座標変換が対称か。
- 勝敗判定がSouth/Northで同じ条件を使っているか。

### 8.5 ペア単位の集計

ランダム開局試験では、単純な全勝率だけでなく、同一開局の2局を1ペアとして次のように分類する。

- Southが2勝
- Northが2勝
- 各1勝
- 引き分けを含む

この方法により、開局そのものの有利不利と座席効果を分離しやすくなる。

## 9. 研究上の注意

AI自己対局の結果は、特定のAIがその局面で選んだ手順の結果である。50%付近ならゲームが公平であるとは限らず、100%でも理論的必勝を意味しない。

一方の勝率が極端な場合は、ゲーム固有の先後差だけでなく、AIの着手順序、乱数消費、評価関数、座標変換、勝敗判定、探索深度などの実装要因を優先的に監査する必要がある。

最終的な主張には、複数開始局面、複数探索深度、鏡像対称性テスト、熟練者の対局記録、可能であれば理論的な局面解析を組み合わせる。
