# J001 — 6番穴・右（応手頑健性で反証）

- 対象phase: `namua`
- 開始局面hash: `f9375bb605c18c7bf0ef4a1f5f5bcce3981380c026b2d7026a184c0302fb856a`
- 手番: South
- 検証状態: `refuted`
- 推奨の強さ: 全4初手比較では首位だったが、全North応手固定試験で応手頑健性を満たさなかったため一般推奨しない

## 盤面図

標準初期局面。数値は各プレイヤーから見た`index 0`〜`7`、`*`はnyumba（index 4）である。

```text
North back   0 0 0 0 0 0 0 0
North front  0 0 0 0 6* 2 2 0
South front  0 0 0 0 6* 2 2 0
South back   0 0 0 0 0 0 0 0
reserve     North 22 / South 22
```

推奨候補:

```text
takata:namua:0:5:right:::false
```

Southから見た6番穴（内部`index 5`）へreserveから1石を加え、右方向へ播く。

## 手の目的

全4合法初手を同じ6 AI条件で終局まで継続した比較において、South勝が3/6で単独首位だったため重点検証した。しかし、Northの全4応手を固定した追加試験では合計7/24勝、最悪応手に対して0/6勝となった。短期探索の保守値では第3位であり、短期値と長期勝敗が一致して選んだ手でもない。

## 基本系列

bao depth 2、legacy depth 2、bao-v2 depth 2の自己対局で一致した冒頭は次のとおり。

```text
1. South takata  index 5 / right
2. North takata  index 5 / right
3. South takata  index 6 / left
4. North capture index 4 / direction right / side left
5. South capture index 7 / direction left / side right / nyumba stop
6. North takata  index 6 / left
```

これは観測系列である。全応手固定試験では、このNorth第2手はSouthが4/6勝した最も良い応手側であり、最悪応手ではなかった。

## 最悪応手

現在の最終勝敗基準では、Northの次手が全6条件でNorth勝となった。

```text
1. South takata index 5 / right
2. North takata index 5 / left
```

この第2手はPhase 1の短期探索値ではbao depth 3を除いて最悪応手と判定されなかった。したがって、応手後の浅い評価値だけでは主要応手を選べない。

## 主要変化

- bao depth 1: Northも`index 5 / right`。South勝、67 ply、`no-move`。
- bao depth 2: 上記基本系列。South勝、65 ply、`front-empty`。
- bao-v2 depth 2: bao depth 2と同一系列・同一終局。South勝、65 ply。
- bao depth 3: Northは`index 6 / right`。North勝、56 ply。
- bao depth 4: Northは`index 6 / left`。North勝、54 ply。
- legacy depth 2: 上記基本系列だが、North勝、54 ply。

## 避けるべき扱い

- C0旧候補`index 5 / left`を、2〜8 ply評価だけを根拠に推奨しない。同一6条件の継続ではSouth 0勝だった。
- J001を「勝率50%」と一般化しない。6局は条件の異なる決定論的自己対局で、反復標本ではない。
- 基本系列を相手の応手に関係なく固定して指さない。depth 3・4ではNorthの第2手から分岐する。
- Northの`index 5 / left`を無視してJ001を一般推奨しない。全6条件でSouthが敗れた。

## 頑健性試験

- 全4初手・同一6条件継続: 3勝3敗、24局中の単独首位。
- 全24局が120 ply以内に終局し、timeoutと打切りは0。
- 全1,399継続着手の合法性、最終state hash、勝者を再生検証済み。
- 絶対支持基準（4/6勝）には未達。
- 全4 North応手固定継続: South 7勝、North 17勝、打切り0。
- 応手別South勝は0/6、4/6、2/6、1/6で、事前基準の各応手3/6以上に未達。
- North `index 5 / left`には全6条件で敗れたため、一般的な定石候補として`refuted`。
