# Bao la Kiswahili 初心者向け上達・基本戦略ガイド

Version: 0.1.0  
更新日: 2026-07-14

## 1. このガイドの目的

このガイドは、Bao la Kiswahiliのルールを一通り理解した人が、実戦で何を考え、どのように練習すればよいかを整理した初心者向け資料である。

Baoでは、1回の着手の中で種まき、捕獲、再種まきが連続する。そのため、定石や長い先読みを覚える前に、まず「1手の結果を正確に読む力」を身につける必要がある。

本書の目標は、次の3点である。

- 毎手の確認手順を固定する。
- 初心者が陥りやすい判断ミスを減らす。
- 対局と局面レビューを上達につなげる。

> [!NOTE]
> Baoには地域差やルール解釈の差がある。本ガイドは、このリポジトリが採用するルールエンジンを前提とする。合法手、捕獲義務、namua、mtaji、nyumbaなどの最終的な判定は、実装されたルールに従う。

## 2. なぜBaoは難しいのか

Baoの難しさは、単に合法手が多いことだけではない。主に次の要素が重なっている。

- 1手の内部で複数回の種まきが連鎖する。
- 最初の見た目から最終盤面を予測しにくい。
- 捕獲量が多い手が、必ずしも良い手ではない。
- 自分の着手後に相手へ強い捕獲を与えることがある。
- 前列の配置、可動性、次の捕獲可能性を同時に考える必要がある。
- namua、mtaji、nyumbaなど、局面の段階や状態によって判断基準が変わる。

したがって、初心者が最初から何手も先を読むのは難しい。まずは、現在の1手を最後まで正確に追跡することが上達の中心となる。

## 3. 毎手の基本確認手順

毎手、次の順番で確認する。

```text
捕獲義務
  ↓
合法手の確認
  ↓
種まきの経路
  ↓
最終停止地点
  ↓
相手の最も強い反撃
  ↓
自分の前列と次の可動性
```

さらに短く覚える場合は、次の3点でよい。

> **取れるか、どこで止まるか、取り返されないか。**

### 3.1 捕獲できる手を最初に確認する

Baoでは、捕獲可能な局面で着手の選択が制約されることがある。最初に自由な候補手を考えるのではなく、次を確認する。

1. 捕獲可能な手があるか。
2. 捕獲義務があるか。
3. その条件下での合法手はどれか。

合法手の分類を誤ると、その後の戦略判断もすべて無効になる。

### 3.2 種数を「距離」として読む

穴の種数は、単なる得点ではない。その穴から種まきを始めたときに、最後の種がどこへ到達するかを決める数値でもある。

たとえば、ある穴に7個の種がある場合は、「7個持っている」と見るだけでなく、「種まきによって7区間先まで盤面を変化させる」と考える。

最初の練習では、着手前に次を答えられるようにする。

- 最初の一周で最後の種はどこに入るか。
- 最後の穴から再種まきが起きるか。
- 捕獲が発生するか。
- 最終的にどこで着手が終了するか。

### 3.3 手の途中ではなく最終盤面を見る

途中で大量に捕獲できても、手の終了時に自分の形が悪化することがある。

確認すべきなのは、途中の最大捕獲数ではなく、手が完全に終了した後の状態である。

- 自分と相手の種数はどう変わったか。
- 自分の前列に空穴が増えていないか。
- 相手に新しい捕獲開始点を与えていないか。
- 自分の次の合法手が減っていないか。
- 次の相手手番で大きな連鎖が発生しないか。

### 3.4 相手の最も嫌な返しを1つ探す

初心者の段階で、相手の全応手を読む必要はない。自分の候補手ごとに、相手の最も強そうな返しを1つ探す。

優先して調べる返しは次のとおりである。

- 最大の捕獲ができる手
- 自分の前列を大きく崩す手
- 自分の合法手や可動性を減らす手
- 連続種まきを長く継続できる手
- 勝敗へ直結する手

「相手が都合のよい手を指す」と仮定せず、最も厳しい返しを基準に判断する。

## 4. すぐに実践できる基本戦略

### 4.1 目先の大量捕獲を信用しすぎない

一度に多くの種を捕獲できる手は魅力的だが、それだけで好手とは限らない。

大量捕獲の後に次の現象が起きる場合がある。

- 相手からそれ以上の反撃を受ける。
- 自分の前列が空になる。
- 相手に有力な捕獲開始点を渡す。
- 自分の種が不利な穴へ偏る。
- 次の手番で動かせる穴が少なくなる。

初心者向けの簡易判断式は次のとおりである。

```text
今回の利益
- 相手の次の反撃
- 自分の前列の損失
- 自分の可動性の損失
```

正確な数値化は不要だが、この4項目を比較する習慣が重要である。

### 4.2 前列を不用意に空にしない

前列は、捕獲、可動性、勝敗条件に関わる重要な領域である。

着手後に次を確認する。

- 前列の空穴が急に増えていないか。
- 種が1か所へ極端に集中していないか。
- 相手と向かい合う穴が無防備になっていないか。
- 次の手番でも前列から有効な着手ができるか。

前列の種は、単なる保有数ではなく、将来の行動力でもある。

### 4.3 相手へ良い捕獲開始点を渡さない

Baoでは、自分の手の終了地点や前列の配置が、相手の捕獲可能性を直接変化させる。

各候補手について、着手後に相手側から盤面を見る。

- 相手はどの穴から捕獲を始められるか。
- 捕獲後の種まきはどちら側から始まるか。
- その連鎖は自分のどの穴を崩すか。
- 相手の合法手数は増えるか。

自分の捕獲だけを見るのではなく、上下で向かい合う穴を一組として見る。

### 4.4 可動性を残す

強い局面とは、種を多く持つ局面だけではない。次の手番で複数の有効な選択肢を持てる局面も強い。

着手後に合法手が1つしか残らない場合、相手に手順を読まれやすくなる。逆に、複数の捕獲候補や種まき方向を残せれば、相手はすべてへ対応しにくい。

候補手を比較するときは、次の手番に残る選択肢の数と質も評価する。

### 4.5 Nyumbaは目的なく崩さない

Nyumbaは通常の穴とは異なる性質を持つ。局面によっては、種まきを継続するか止めるかという選択や、特別な防御・展開上の価値を持つ。

初心者の段階では、次の問いに答えられない場合、目的なくnyumbaを崩さない方が安全である。

- 特別な性質を失っても、十分な利益があるか。
- その手で具体的な捕獲や防御が成立するか。
- 相手の反撃まで考慮して有利か。

ただし、nyumbaの温存は絶対則ではない。局面上の具体的な利益を優先する。

## 5. 候補手の比較方法

最初からすべての合法手を詳細に比較すると負担が大きい。まず有力そうな2手へ絞り、次の観点で比較する。

| 評価項目 | 確認内容 |
| --- | --- |
| 捕獲 | 今回どれだけ取れるか |
| 反撃 | 相手にどれだけ取り返されるか |
| 前列 | 自分の前列を維持できるか |
| 可動性 | 次の手番にも有効な合法手が残るか |
| 連鎖 | 次の捕獲や種まきへつながるか |
| 強制力 | 相手の選択肢をどれだけ制限できるか |

簡易的に各項目を`良い・普通・悪い`で評価してもよい。重要なのは、捕獲数だけで候補手を決めないことである。

## 6. 段階別の上達訓練

### 第1段階：合法手を間違えない

最初の目標は勝利ではなく、ルール上正しい候補手を毎回列挙できることである。

重点項目:

- 捕獲義務の有無
- namuaとmtajiの違い
- 種まき方向
- 捕獲後の種の投入位置
- 連続種まきの継続条件と終了条件
- nyumbaの状態と扱い

ゲームでは「やさしい」など低い難易度を使用し、着手前に合法手を自分で数えてから操作する。

### 第2段階：第1着地点を必ず当てる

任意の合法手を選び、実際に操作する前に、最初の種まきがどこで終わるかを予測する。

最初から長い連鎖全体を読む必要はない。

```text
第1着地点を当てる
  ↓
再種まき1回後まで読む
  ↓
捕獲後まで読む
  ↓
手の完全な終了まで読む
```

予測後にゲーム上で着手し、結果と照合する。

### 第3段階：1手を最後まで盤上で動かさずに読む

着手前に、次を紙または頭の中で予測する。

- 種まきの経路
- 捕獲される穴
- 再種まきの開始位置
- 最終停止地点
- 手番終了時の主な穴の種数

1局を速く終えるより、1手の予測と答え合わせを繰り返す方が、この段階では効果的である。

### 第4段階：相手の次の最善手を1つ探す

自分の着手を実行する前に、相手の最も厳しい返しを1つ選ぶ。

その返しによって、自分の候補手の評価が大きく下がる場合は、別の候補手を検討する。

この訓練により、自分側だけを見る一方向の読みから、相手の反撃を含む読みへ移行できる。

### 第5段階：候補手を2つ比較する

候補手Aと候補手Bについて、次の情報を記録する。

```text
候補手A
- 自分の捕獲:
- 相手の最大反撃:
- 着手後の前列:
- 次の可動性:

候補手B
- 自分の捕獲:
- 相手の最大反撃:
- 着手後の前列:
- 次の可動性:
```

結果だけでなく、着手前の予測が正しかったかを振り返る。

### 第6段階：短い先読みへ進む

1手の最終盤面を安定して読めるようになったら、次の順序で読みを伸ばす。

1. 自分の候補手
2. 相手の最善と思われる返し
3. その後の自分の応手

探索用語では、概ね3半手分の応酬を意識する段階である。ただし、Baoでは1回の着手自体に長い連鎖が含まれるため、手数を増やすより各着手の正確性を優先する。

## 7. 対局後の復習

すべての着手を見直す必要はない。1局につき、次の中から最大3局面を選ぶ。

- 大量捕獲を受ける直前の局面
- 自分が長く迷った局面
- AIが長く考えた局面
- 大量捕獲をした直後に反撃された局面
- 急に合法手が減った局面
- 勝敗が決まった直前の局面

各局面では次を確認する。

1. 他にどの合法手があったか。
2. 自分は最終停止地点を正しく予測できていたか。
3. 相手の最大反撃を見落としていなかったか。
4. 前列と可動性はどう変化したか。
5. 結果ではなく、着手時点の判断にどの問題があったか。

このリポジトリの診断機能を使う場合は、調べたい局面やAI着手をJSONとして保存し、後から再現できる。詳細は[`AI_HUMAN_REVIEW_GUIDE.md`](AI_HUMAN_REVIEW_GUIDE.md)を参照する。

## 8. AI対戦を使った練習メニュー

### 8.1 「やさしい」：ルールと計算精度

目的:

- 合法手を正しく見つける。
- 第1着地点を当てる。
- 1手の連鎖を最後まで追う。

勝敗より、予測の正確性を記録する。

### 8.2 「ふつう」：反撃確認

目的:

- 自分の着手後に相手の強い返しを予測する。
- 目先の捕獲と反撃後の収支を比較する。
- 候補手を2つ比較する。

### 8.3 「むずかしい」：局面レビュー

目的:

- AIと自分の候補手が異なる局面を保存する。
- 深い探索で選ばれた手の結果を確認する。
- 自分が見落とした反撃や可動性の変化を探す。

AIの手を無条件に正解とみなさない。探索深度、timeout、評価関数の限界があるため、AIの着手は比較対象として扱う。

### 8.4 「ムタアラム」：重要局面の比較

目的:

- 同じ局面で、より長い探索による着手変化を見る。
- 「むずかしい」と選択手が異なる局面を調査する。
- 候補手の評価が深度によって変わる難局を集める。

## 9. 初心者が避けたい典型的な誤り

### 9.1 最初に見つけた捕獲手をすぐ指す

捕獲義務があっても、捕獲可能な手が複数ある場合がある。それぞれの最終盤面と反撃を比較する。

### 9.2 捕獲数だけで手を評価する

捕獲量、前列、可動性、反撃可能性を合わせて評価する。

### 9.3 種まきの途中で局面評価を止める

連続種まきが完全に終了するまで、着手の結果は確定しない。

### 9.4 相手側の盤面を見ない

自分の着手後は、盤面を相手側から見直し、相手の捕獲可能手を確認する。

### 9.5 長い先読みを急ぎすぎる

1手の計算が不正確なまま3手先を読んでも、分岐の土台が誤っている。まず1手を正確に読む。

### 9.6 負けた最後の手だけを反省する

敗因は、最終手より前に前列や可動性を失った局面にあることが多い。大きく形勢が変わった最初の局面を探す。

## 10. 実戦用チェックリスト

着手前に次を確認する。

- [ ] 捕獲義務を確認した。
- [ ] 合法手を複数確認した。
- [ ] 最初の種まきの着地点を予測した。
- [ ] 再種まきと捕獲を最後まで追った。
- [ ] 相手の最大反撃を1つ確認した。
- [ ] 自分の前列が大きく減らないことを確認した。
- [ ] 次の手番の可動性を確認した。
- [ ] 目先の捕獲数だけで選んでいない。

初心者のうちは、すべてを毎回行う必要はない。最初は次の3項目だけでもよい。

- [ ] 捕獲できるか。
- [ ] 最終的にどこで止まるか。
- [ ] 相手に大きく取り返されないか。

## 11. 上達の目安

| 段階 | 到達目標 |
| --- | --- |
| 入門 | 捕獲義務と合法手を正しく判断できる |
| 初級1 | 第1着地点を安定して予測できる |
| 初級2 | 1手の連鎖を最後まで追える |
| 初級3 | 相手の最大反撃を1つ予測できる |
| 中級入口 | 候補手を前列、可動性、反撃で比較できる |
| 中級 | 自分、相手、自分の短い応酬を読める |

勝率だけで上達を評価しない。次の変化も重要な進歩である。

- 合法手の見落としが減った。
- 種数の数え直しが減った。
- 着手前の最終盤面予測が当たるようになった。
- 大量捕獲後の反撃を事前に発見できた。
- 敗因となった局面を自分で特定できた。

## 12. 最初に覚える5原則

1. **捕獲できる手を最初に確認する。**
2. **手の途中ではなく、完全な終了後の盤面を見る。**
3. **取った数より、相手の反撃を重視する。**
4. **自分の前列を不用意に空にしない。**
5. **穴の種数を、着地点までの距離として読む。**

Baoの上達で最初に必要なのは、長い先読みや定石暗記ではない。

> **1手の連鎖を間違えずに読み切る計算精度が、すべての戦略の土台となる。**
