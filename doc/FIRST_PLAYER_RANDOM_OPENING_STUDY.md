# Bao la Kiswahili ランダム開局における先後差調査

Version: 0.1.0  
更新日: 2026-07-14

## 1. 目的

標準初期局面からの決定的な自己対局だけでは、Bao全体の先攻・後攻差を評価できない。本調査では、namuaおよびmtajiの途中までランダムに進めた局面を多数生成し、その局面で手番を持つSouthと、後手側のNorthの勝率を比較した。

本調査が対象とするのは、このリポジトリのルール実装とAIによる実験結果である。Bao la Kiswahiliの理論的な先手必勝・後手必勝を証明するものではない。

## 4. phase別集計

### 4.1 namua

| 指標 | 結果 |
| --- | ---: |
| 対局数 | 600局 |
| South勝利 | 292局 |
| North勝利 | 308局 |
| South勝率 | 48.7% |
| 95% Wilson信頼区間 | 44.7%–52.7% |

信頼区間に50%を含むため、namuaのランダム局面全体では明確な先攻優位または後攻優位を検出できなかった。

8手開局だけではNorthが57%勝利したが、4手および12手ではほぼ均衡している。したがって、namuaに一貫した後攻優位があるとはいえず、特定の局面分布またはサンプル変動に依存した可能性がある。

### 4.2 mtaji

| 指標 | 結果 |
| --- | ---: |
| 対局数 | 600局 |
| South勝利 | 344局 |
| North勝利 | 256局 |
| South勝率 | 57.3% |
| 95% Wilson信頼区間 | 53.3%–61.2% |

信頼区間が50%を上回ったため、この実験条件ではmtajiの手番側に統計的に確認できる優位が観測された。

4手・8手・12手のすべてでSouth勝率が56%以上であり、開局手数を変えても方向が一致している。ただし4手・8手の個別条件では信頼区間がわずかに50%を含む。最も明確なのは12手条件の60.0%である。

## 5. 全体集計の注意

6条件を単純合算すると次の結果になる。

| 指標 | 結果 |
| --- | ---: |
| 対局数 | 1,200局 |
| South勝利 | 636局 |
| North勝利 | 564局 |
| South勝率 | 53.0% |
| 95% Wilson信頼区間 | 50.2%–55.8% |

しかし、この53.0%をBaoの一般的な先攻勝率として扱ってはならない。namuaとmtajiは異なるゲーム段階であり、本調査では両phaseへ同数の600局を人為的に割り当てている。実戦における局面分布とは一致しないためである。

## 6. 現時点の解釈

今回の結果は、Baoの先後差を単一の固定値で表すことが難しいことを示している。

- 標準初期局面の深度2以上では後攻Northが勝つ決定的系列が観測された。
- 多様化したnamua局面600局では、先後差はほぼ均衡した。
- 多様化したmtaji局面600局では、手番側Southに約57%の優位が観測された。

したがって、現段階で最も妥当な仮説は次のとおりである。

> Bao la Kiswahiliにはゲーム全体を通じた単純な先攻・後攻優位ではなく、局面構造とphaseに依存する手番価値が存在する可能性が高い。特にmtajiでは、現在手番を持つ側のテンポが勝率へ強く影響している可能性がある。

ただし、標準初期局面から始めた対局における最終的な「先攻プレイヤーの勝率」を解明するには、開局をランダム化するのではなく、複数の強さ・探索方式・評価関数で標準初期局面から対局させる必要がある。

## 7. 次の検証

次の優先順位で調査を継続する。

1. 標準初期局面を深度2・3・4で、異なる評価プロファイルおよび探索方式を使って比較する。
2. AIの決定的な同型対局を避けるため、上位候補から確率的に着手する自己対局を実施する。
3. 標準初期局面から生成した多様な序盤系列について、先攻・後攻を維持したまま最終勝率を測定する。
4. 同一局面をプレイヤー反転した鏡像局面で評価し、ルールエンジンと評価関数の対称性を検証する。
5. 可能であれば人間の熟練者対局記録を収集し、AI実験との整合性を確認する。

## 8. 結論

> ランダム化したnamua局面では明確な先後差は確認されなかった。一方、mtaji局面では手番側が600局中344勝、勝率57.3%となり、統計的に確認できる手番優位が観測された。したがって、Baoの先後差は固定的ではなく、少なくともゲームphaseと局面分布に依存すると考えられる。
